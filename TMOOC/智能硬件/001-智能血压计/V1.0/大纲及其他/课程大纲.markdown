# 基于STM32的智能血压计开发 #
以智能血压计开发为主线，讲解相关的嵌入式、物联网开发技术。这门课程不是零基础的，要求学习者具备以下基础：	

- 熟练使用计算机、了解计算机软件
- 有软件开发的基本概念
- 有扎实的 C 语言基础

一个小节7分钟左右 一课5小节左右 共半小时左右
